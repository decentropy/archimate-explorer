ArchiMate Explorer

If you've just downloaded this, it's initialized with Archinsurance model. Try a search and explore, then read below to import your own model.

Use your own model

  1. From Archi, export "Open Exchange" XML into 'static/data' folder
  2. From Archi, generate HTML report into 'static/reports' folder
  3. Edit config section of flaskapp.py with your own values.
  4. Then click here to process the data.
